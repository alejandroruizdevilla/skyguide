# Flight Simulator Images

This directory should contain images for each flight simulator.

## Required Images

Add the following image files (JPG format recommended):

### Professional Simulators
- `msfs2024.jpg` - Microsoft Flight Simulator 2024
- `xplane12.jpg` - X-Plane 12
- `prepar3d.jpg` - Prepar3D v6
- `aerofly.jpg` - Aerofly FS 4

### Combat Simulators
- `dcs.jpg` - DCS World
- `il2.jpg` - IL-2 Sturmovik
- `falcon-bms.jpg` - Falcon BMS

### Free & Open Source
- `flightgear.jpg` - FlightGear
- `ysflight.jpg` - YSFlight

### Mobile & Casual
- `infinite-flight.jpg` - Infinite Flight
- `rfs.jpg` - RFS - Real Flight Simulator

### Arcade & Gaming
- `war-thunder.jpg` - War Thunder
- `ace-combat-7.jpg` - Ace Combat 7
- `roblox.jpg` - Roblox Flight Sims

### Specialty
- `condor2.jpg` - Condor 2
- `fsx.jpg` - Microsoft FSX

## Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: 800x450 pixels (16:9 ratio) recommended
- **Size**: Keep under 200KB per image for fast loading

## Where to Find Images

You can get screenshots or logos from:
- Official game websites
- Steam store pages
- Official press kits
- In-game screenshots (if you own the games)

Make sure you have the right to use any images you add!
